# Salesforce Commerce 

---------------------
Tamara’s Pre-Check API is a real-time tool that determines shopper eligibility before they even select a payment method, ensuring you only show Tamara to those most likely to convert.

By Implementation

Eligible Shoppers: See Tamara and enjoy a high-confidence conversion.
Ineligible Shoppers: Tamara is hidden or greyed out, eliminating guaranteed declines and keeping the customer in the purchase funnel to try another payment method.
The Business Impact

Conversion Improvement: Typically, we see a ~5% improvement in KSA and a ~10% improvement in UAE.
Efficiency: The API has a latency of <100ms, ensuring zero measurable impact on your checkout performance.
